排序演算法詳細介紹 🔗

氣泡排序 Bubble Sort 🔗

氣泡排序是一種簡單的排序演算法，原理是依序比較相鄰兩個元素，如果前一個元素大於後一個元素，就交換它們的位置，直到排序完成。因為排序過程中較大的元素會慢慢浮到序列右端，所以稱為氣泡排序。

動畫演示 🔗

實作步驟 🔗

  1. 外層 for 控制待排序元素數量，每輪減少一個，直到排序完成。
  2. 內層 for 表示每輪需要比較的次數。
  3. 若前一個數大於後一個數，則使用 swap() 交換。

C++ 範例 🔗

優化 🔗

若某輪沒有交換，代表數列已排好，可提前結束，減少不必要比較。

時間複雜度 🔗

  • O(n2)O(n^2)

總結 🔗

簡單但效率低，適合小規模數據。

選擇排序 Selection Sort 🔗

選擇排序原理是每輪選擇最小元素與當前位置交換，直到排序完成。

動畫演示 🔗

實作步驟 🔗

  1. 外層 for 遍歷 0on−10 o n-1。
  2. 設最小值索引為 i。
  3. 內層遍歷 i 後元素，更新最小值位置。
  4. 交換第 i 個元素與最小值。

C++ 範例 🔗

時間複雜度 🔗

  • O(n2)O(n^2)

總結 🔗

簡單、效率低，通常只用於小型數據，且不穩定。

插入排序 Insertion Sort 🔗

將未排序元素依序插入已排序序列。

動畫演示 🔗

C++ 範例 🔗

注意 🔗

  • while 條件需檢查邊界，插入點須 ≥0\ge 0。

時間複雜度 🔗

  • O(n2)O(n^2)

總結 🔗

簡單、效率低，適合小規模數據。

合併排序 Merge Sort 🔗

分治法排序，先拆分再合併。

C++ 範例 🔗

時間＆空間複雜度 🔗

  • O(nlog⁡n)O(n \log n)
  • 空間 O(n)O(n)

總結 🔗

高效穩定，適合大量數據，需注意邊界。

快速排序 Quick Sort 🔗

分治法排序，選基準元素 pivot 分兩半排序。

C++ 範例 🔗

時間複雜度 🔗

  • 平均 O(nlog⁡n)O(n\log n)，最壞 O(n2)O(n^2)

注意 🔗

  • 不穩定，選 pivot 影響效率，可用隨機 pivot。

總結 🔗

高效排序，適合大數據，最壞情況較差。

堆排序 Heap Sort 🔗

使用二元樹構建最大堆排序。

C++ 範例 🔗

時間＆空間複雜度 🔗

  • O(nlog⁡n)O(n\log n)，空間 O(1)O(1)

總結 🔗

高效排序，原地排序，常數因子較大。

希爾排序 Shell Sort 🔗

分組插入排序，逐步縮小間隔。

C++ 範例 🔗

時間＆空間複雜度 🔗

  • 平均/最壞 O(n(log⁡n)2)O(n (\log n)^2)，最佳 O(n)O(n)，空間 O(1)O(1)

總結 🔗

比插入排序快，適合大數據。

計數排序 Counting Sort 🔗

適合範圍小整數排序。

C++ 範例 🔗

注意 🔗

  • 適合範圍小整數，不適合浮點數或負數。穩定排序。

時間＆空間複雜度 🔗

  • 時間 O(n+k)O(n+k)，空間 O(k)O(k)

桶排序 Bucket Sort 🔗

將資料分桶，再對桶內排序。

C++ 範例 🔗

注意 🔗

  • 桶大小、數量影響效率與空間。

時間＆空間複雜度 🔗

  • 時間 O(n+k)O(n+k)，空間 O(n+k)O(n+k)

基數排序 Radix Sort 🔗

非比較排序，按位排序。

C++ 範例 🔗

注意 🔗

  • 適合整數，使用穩定排序。

時間＆空間複雜度 🔗

  • 時間 O(d(n+k))O(d(n+k))，空間 O(n+k)O(n+k)
